- Chords bridge: DbMaj7 / Gb7 / DbMaj7 / Gb7 / Ami7 / D7 / Gmi7/ C7
- Some voice leading tricks you can use:
- DbMaj7 (Db F Ab C) => Gb7 (Fb Gb Bb C)
- Ami7 (E G A C) => D7 (D F# A C) => Gmi7 (D F G Bb) => C7 (C E G Bb)
